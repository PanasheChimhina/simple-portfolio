Trading Journal
@ HoyoTrader

HoyoTrader is a trading journal with integrated analytics that saves and backs up trades executed by (mainly) stock, forex and crypto currency traders. Users can manualy enter their trades and the HoyoTrader software carries out a thorough analysis of all the trades and issues back graphical representations of the analysis report. Users can use this insightful information to enhance their trading strategies.

Core Technologies:
React, CSS, MongoDB, HTML, Vanilla JavaScript, Node, Express, D3.

Plagiarism Checker
@ Similarity Checker

A web application that checks the similarity between documents. It primarily consists of two types of checks namely Standard check and Web check. The standard check carries out a plagiarism check between two documents and the web check makes a google based search using every sentence in the given document. Both checks return a similarity report which highlights the percentage similarity.

Core Technologies:
Express, Node, EJS, HTML, CSS, Vanilla JavaScript.

English Translator
@ American / British English Translator

A web application that translates American English to British English or Vice versa. The app handles numerous varying aspects of both types of English such as word spellings, grammar, nouns, phrases titles as well as time representations. Users can enter British/American text and the app translates it.

Core Technologies:
Express, Node, HTML, CSS, Vanilla JavaScript.
